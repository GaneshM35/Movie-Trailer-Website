This is Movie Trailer Website Project Source
This folder contains THREE python files in it.

1)media.py -- contain MovieTrailer class.
2)entertainment_center -- contacts list of movie objects.
3)fresh_tomatoes -- contains the open_movies_page() function that will take in your list of movies and generate an HTML file including this content, producing a website to showcase your favorite movies.


To View the Movie Trailer Website
run the entertainment_center.py file.
